## Description

This is a Register/Login APIs using NestJS, PostgreSQL, and Prisma.

## Requirements

1. Register
2. Login
3. Return a user

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

## Rest APIs guide

1. Register: [POST] .../auth/register
2. Login: [POST] .../auth/login
3. Return a user: [GET] /users/:id

## Technical Stacks
- NestJS
- TypeScript
- PostgreSQL
- Prisma
